# Site Eduardo Pitta

Site memorial de Eduardo Pitta (1949–2023), construído em HTML/CSS/JS estático para alojamento no GitHub Pages.

## Estrutura de ficheiros

```
eduardopitta-site/
├── index.html          ← site completo (todas as páginas)
├── CNAME               ← domínio eduardopitta.pt
├── css/
│   └── style.css       ← todo o design e responsivo
└── images/             ← copiar imagens do servidor actual
    ├── logo1.png       ← assinatura (de eduardopitta.pt/images/logo1.png)
    ├── ed02.jpg        ← foto da entrada PT (de eduardopitta.pt/images/2021/04/30/ed02.jpg)
    └── [outras]        ← capas dos livros, etc.
```

## Imagens a copiar do servidor actual

Antes de publicar, descarregar e guardar na pasta `images/`:

| Ficheiro local     | URL original |
|--------------------|--------------|
| `logo1.png`        | https://eduardopitta.pt/images/logo1.png |
| `ed02.jpg`         | https://eduardopitta.pt/images/2021/04/30/ed02.jpg |
| Capas dos livros   | https://eduardopitta.pt/images/2021/05/04/ |

## Como publicar no GitHub

1. Abrir o GitHub Desktop
2. Seleccionar o repositório `eduardopitta-site`
3. Arrastar estes ficheiros para a pasta do repositório
4. Escrever uma mensagem (ex: "Novo site") e clicar **Commit to main**
5. Clicar **Push origin**

O site fica disponível em https://eduardopitta.pt em alguns minutos.

## Como editar conteúdo

Todo o conteúdo está no ficheiro `index.html`.
Abrir com qualquer editor de texto (ex: Notepad, TextEdit).
Procurar o texto a alterar, editar, guardar e publicar novamente.

## Páginas

- **Entrada** — bio, foto, poema, assinatura
- **Obra** — lista completa de publicações
- **Crítica** — publicações periódicas
- **Memorial** — a completar com conteúdo do Facebook e outros
- **Arquivo** — Entrevistas / Imprensa / Publicações Dispersas
